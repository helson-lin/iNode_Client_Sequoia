本项目为教程：二次封装 inodeClinet 支持Macos15.

当然同类型的 pkg的安装包（非 flat）可以通过相同方式二次封装。



## 📦 二次封装支持Macos15

首先需要下载[Packages](http://s.sudre.free.fr/Software/Packages/about.html)软件。

`git clone`本项目之后，双击`fkInode.pkproj`使用 Packages软件打开。
